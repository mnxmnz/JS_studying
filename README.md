## 배열
* 배열에 데이터 한 개 추가하기
<pre>
<code>
var li = ['a', 'b', 'c', 'd', 'e'];
li.push('f');
alert(li);
</code>
</pre>
* 배열에 데이터 여러 개 추가하기
<pre>
<code>
var li = ['a', 'b', 'c', 'd', 'e'];
li = li.concat('f', 'g', 'h');
alert(li);
</code>
</pre>
* 배열의 시작점에 원소 추가하기
<pre>
<code>
li.unshift('z');
alert(li);
</code>
</pre>
