## 배열
* 배열에 데이터 한 개 추가하기
* var li = ['a', 'b', 'c', 'd', 'e'];
<pre>
<code>
li.push('f')
</code>
</pre>
* 배열에 데이터 여러 개 추가하기
<pre>
<code>
li = li.concat('f', 'g', 'h');
</code>
</pre>
* 배열의 시작점에 데이터 추가하기
<pre>
<code>
li.unshift('z');
</code>
</pre>
